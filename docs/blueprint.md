# **App Name**: 3D Portfolio X

## Core Features:

- 3D Portfolio Showcase: Visually stunning 3D-rendered portfolio pieces.
- Admin Panel: Secure admin interface to manage all portfolio content.
- Portfolio Project Management: Add, edit, and remove portfolio projects with descriptions, images, and 3D models.
- Blog System: Publish blog posts related to graphic design and 3D art to showcase expertise.
- Contact Inbox: Receive and manage messages from potential clients and collaborators directly within the admin panel.
- SEO Tools: Built-in SEO tools to optimize content for search engines.

## Style Guidelines:

- Primary color: Dark Indigo (#4B0082) to evoke sophistication and depth within the dark theme.
- Background color: Near-black (#121212) for a modern, immersive experience, offering a canvas that highlights the 3D portfolio.
- Accent color: Electric Purple (#BF00FF) for interactive elements, creating a dynamic visual gradient.
- Body and headline font: 'Space Grotesk', a proportional sans-serif font well suited to the 3D portfolio's need for modernity.
- Use sleek, minimalist icons for navigation and interaction.
- Grid-based layout with asymmetrical elements to enhance visual appeal and balance.
- Smooth transitions and subtle 3D animations to guide user interaction and enrich the dark, gradient theme.
